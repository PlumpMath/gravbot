gravbot
=======

This is a little top down shooter using the Panda3D framework

## Features

- Destructible world
- Zero gravity
- Chunks of world float around
- Procedural generation
- Terrible AI
- Broken gun physics

## Instructions

Install Panda3D

Clone the game:

    git clone https://github.com/michaeltchapman/gravbot
    cd gravbot

run the game using:

    ppython launch.py
